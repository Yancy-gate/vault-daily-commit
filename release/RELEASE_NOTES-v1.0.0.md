# vault-daily-commit v1.0.0

Obsidian 每日笔记**安全 Git 提交** Agent Skill — 跨 Cursor / Claude Code / OpenCode / Codex。

路径与提示词请以当前仓库 `vault-config.md` / `prompts.md` 为准（通用占位，安装后自行配置）。

## 下载哪个包？

| 压缩包 | 适合谁 |
|--------|--------|
| **universal** | 自己选平台 |
| **cursor** / **claude-code** / **opencode** / **codex** | 对应 Agent |

每个平台包内有 **`INSTALL.md`**。

## 最短提示词

```text
帮我按 vault-config 提交今天的笔记和附件截图，别带无关文件，先 status 再 commit，不要 push。
```
